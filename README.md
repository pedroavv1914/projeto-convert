## ✨ Features
- 💵 Convert USD, EUR, and GBP to BRL
- 🔢 Input validation (numbers only)
- 🎨 Modern UI with smooth animations
- 📱 Fully responsive design
- ⚡ Instant conversion results

## 📊 Exchange Rates
The app uses fixed exchange rates (as of development):

- 1 USD = 5.54 BRL
- 1 EUR = 6.42 BRL
- 1 GBP = 7.54 BRL
